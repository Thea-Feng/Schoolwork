Under the folder of compile_run.sh, enter  " bash compile_run.sh" to see the result.

Change parameters: The code (main.cpp) provides two groups of parameters, whose results are shown in the report. You can change the annotation to see corresponding results. You can also enter your own parameters and run the code.

Extra Work: you can replace the parameter line_color with a triple (0 < each number < 255) to change the color of the line.
